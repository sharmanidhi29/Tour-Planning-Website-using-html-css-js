# Tour Planning Website

A responsive and interactive **Tour Planning Web Application** built with **HTML, CSS, Bootstrap, and JavaScript**.
This project allows users to explore destinations, create an itinerary, save plans locally, and simulate trip bookings ‚Äî all in one clean and modern interface.

## Features

* **Responsive UI** using Bootstrap 5
* **Destination search & filter** (by type: beach, mountain, city)
* **Interactive destination cards** with details
* **Itinerary builder**

  * Add/remove destinations
  * Saves automatically using `localStorage`
  * Export itinerary as JSON
* **Booking modal** (client-side demo)
* **Contact form**
* **Google Maps embed**
* Clean, accessible design with focus states
* Fully functional in a single HTML file (no backend required)
